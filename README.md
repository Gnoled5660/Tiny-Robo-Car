# Description
This project is a self autonomous driving car. It is using only a camera sensor to navigate. The idea was to build a car as small as I can, which still has a steering joint, so that it can manouver correctly. 

# Motivation
I have always wanted to build a car with proper steering joints. But I also came up with the idea, to build something tiny. Thats why I decided to build a tiny car. For me this was kind of a new challenge, since I have never made something so small, that I had to worry about space. Through out this project, I have learned to build proper steering joints and also cars in general, which I didn't know how to do before.
